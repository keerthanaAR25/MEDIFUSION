# MEDIFUSION: A MULTIMODAL AI SYSTEM FOR CLINICAL  DIAGNOSIS USING SPEECH, IMAGING, OCR, AND LARGE LANGUAGE MODELS

## Multimodal AI Healthcare Framework using Deep Learning, RAG, and LLaMA 3.1

MediFusion is a next-generation multimodal AI healthcare framework that combines manual deep learning model engineering with advanced Large Language Model reasoning to simulate real-world intelligent clinical workflows. The system integrates medical image analysis, speech processing, OCR-based report extraction, Retrieval-Augmented Generation (RAG), and multilingual AI-driven healthcare assistance into a unified diagnostic ecosystem.

Unlike conventional healthcare AI projects that rely solely on pretrained APIs, MediFusion involves independently building, training, testing, and evaluating specialty-specific deep learning models across multiple medical domains including Pulmonology, Neurology, Cardiology, Surgery, Endocrinology, and General Medicine using real medical datasets and custom preprocessing pipelines.

The framework further integrates LLaMA 3.1 for contextual medical reasoning, multimodal fusion analysis, discharge summary generation, multilingual patient guidance, and intelligent clinical interpretation. Audio symptom analysis using Whisper ASR, OCR-based lab report extraction using Tesseract OCR, imaging-based disease prediction, and clinical note interpretation are combined to replicate practical hospital diagnostic workflows.

# Installation

## Clone Repository

```bash
git clone https://github.com/keerthanaAR25/MEDIFUSION.git
cd MEDIFUSION
```

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Backend Setup

```bash
cd backend
pip install -r requirements.txt
```
